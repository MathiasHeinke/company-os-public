# Command EVE Site

Static public preview for Command EVE.

## Source Layout

- `public/` is the deployable source snapshot.
- `dist/` is generated by `npm run build` and is intentionally ignored.
- `vercel.json` defines the Vercel build/output contract and static cache headers.

## Local Commands

```bash
npm run build
npm run verify
npm run preview
```

## Deployment

Preview:

```bash
npm run deploy:preview
```

Production:

```bash
npm run deploy:production
```

For a Git-connected Vercel project, set the Vercel root directory to:

```text
assets/brand/eve-command/site
```

## Current Source Status

This site currently preserves the approved 0.6.8 visual state as a static
source snapshot. The next cleanup step is to replace the compiled landing bundle
with first-class editable React/Remotion source once the cinematic art direction
is stable enough to justify that rebuild.
